# ✨ Spotlight Cursor Effect

A modern and lightweight **spotlight cursor effect** built with vanilla HTML, CSS, and JavaScript.  
This project creates a smooth glowing light that follows the mouse movement, inspired by Apple-style UI interactions.

---

## 🚀 Demo

Move your mouse over the section to see the spotlight effect in action.


---

## 🛠️ Built With

- HTML5
- CSS3 (Custom Properties)
- JavaScript (Vanilla)

---

## 🎯 Features

- Smooth cursor tracking
- Lightweight & fast
- Fully responsive
- Easy to customize colors & intensity
- Apple-style modern glow effect
- No libraries required
